# Chrome-Icon
本项目旨在提供高分辨率的 Chrome 浏览器默认头像。通过 AI 图像增强技术，原有的 Chrome 默认头像可以被放大并提升图像质量。

## 功能介绍

- **Chrome 默认头像**：提供 Chrome 默认的头像图像集合。
- [默认大小](theme/1x)
- [4倍放大](theme/4x)

## 使用技术

- **Inpaint-web**：基于深度学习的 AI 图像增强工具，支持修复图像、移除不需要的对象和提升图像分辨率等功能。

## 参考
